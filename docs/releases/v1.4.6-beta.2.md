# Release v1.4.6-beta.2

[![GitHub Downloads](https://img.shields.io/github/downloads/alexdelprete/ha-abb-fimer-pvi-vsn-rest/v1.4.6-beta.2/total?style=for-the-badge)](https://github.com/alexdelprete/ha-abb-fimer-pvi-vsn-rest/releases/tag/v1.4.6-beta.2)

**Release Date:** 2026-03-29

**Type:** Beta release - Incremental fix for known devices feature.

## What's Changed (since beta.1)

### Fixed

- **Update device_type in known_devices from discovery** - The v8 migration populates
  `known_devices` from the device registry but sets `device_type` to `"unknown"` for
  non-datalogger devices (only dataloggers can be inferred from the model field). Now on
  each successful discovery, existing `known_devices` entries get their `device_type`
  updated from the actual discovery result (e.g., `"unknown"` → `"inverter_3phases"`).
